    # Synapse
    Bienvenue sur la documentation de la version trois de la librairie de composants du design system de la Caisse Nationnal de L'Assurance Maladie

    ## Introduction

    Cette librairie de composants est basée sur le framework [VueJs](https://vuejs.org/). Et est un superset de [Vuetify](https://vuetifyjs.com/).
    Elle est destiné à combler les besions de la CNAM en terme de composants graphiques.
    
    ## Les différentes versions

    Cette librairies est destiné aux nouveau projet démarrant avec Vue3. Elle est destiné à être compatible avec VueJs ainsi que Nuxt et le rendu coté serveur.

    Les projets utilisant Vue2 doivent se référer à la documentation de la librairie [VueDot](https://digital-design-system.netlify.app/demarrer/introduction/).
    Les projets migrants de Vue2 à Vue3 doivent se référer à la documentation de la librairie [VueDot](https://digital-design-system.netlify.app/demarrer/introduction/) et utiliser le package [bridge](@cnamts/synapse-bridge).

    ## Installation

    Cette librairies n'est pas encore publié sur npm. Pour consulter son avancement, vous pouvez vous référer à la [page github](https://github.com/assurance-maladie-digital/design-system-v3).
