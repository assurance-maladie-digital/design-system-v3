# DownloadBtn

L'élément `DownloadBtn` est utilisé pour permettre à l'utilisateur de télécharger un document provenant d'une API.

<Canvas of={DownloadBtnStories.Default} />

# API

<Controls of={DownloadBtnStories.Default} />

# Exemple d'utilisation

<Source dark code={`
<script setup lang="ts">
	import DownloadBtn from '@/components/DownloadBtn/DownloadBtn.vue'
	import axios from 'axios'

	function download() {
		return axios.get('https://run.mocky.io/v3/63e571d5-1134-4f51-82ac-fa7cc8045124')
	}
</script>

<template>
	<div>
		<DownloadBtn
			:file-promise="download"
			:btn="{ color: 'primary'}"
			@error="console.error"
			@success="console.info"
		>
			Download
		</DownloadBtn>
	</div>
</template>
`} />


# Personnalisation

Vous pouvez utiliser la props `vuetifyOptions` pour personnaliser les composants Vuetify utilisés en interne dans le composant `DownloadBtn`.

Les clés de cette props correspondent aux noms des composants Vuetify et les valeurs aux options que vous souhaitez leur passer.
Les clés autorisées sont `btn` et `icon`.
