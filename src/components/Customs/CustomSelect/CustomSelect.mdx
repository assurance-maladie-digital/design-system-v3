# CustomSelect

Le composant `CustomSelect` est utilisé pour proposer une alternative au `v-select` de Vuetify qui ne respecte pas les règles RGAA.

<Canvas of={CustomSelectStories.Default} />

>Format des items :
>
> - Si les items passes en props sont des objets, il est possible de personnaliser les champs `text` et `value`  via les props `textKey` et `valueKey` (cf ci-dessous).
> - Si les items sont un tableau de string, le composant les utilisera directement.
