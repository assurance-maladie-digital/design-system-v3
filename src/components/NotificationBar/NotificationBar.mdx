# NotificationBar

Le composant `NotificationBar` est utilisé pour afficher des notifications à l’utilisateur.

# Exemple d'utilisation

<Source dark code={`
<template>
  <div class="d-flex flex-wrap align-center justify-center">
    <NotificationBar v-model="showNotification" type="success" :closeBtnText="Fermer">
      <template #default>This is a success notification</template>
    </NotificationBar>

    <VBtn v-if="!showNotification" color="primary" @click="showNotification = true" class="ma-6">Réinitialiser</VBtn>
  </div>
</template>

<script setup lang="ts">
import NotificationBar from '@/components/NotificationBar/NotificationBar.vue'
import { ref } from 'vue'

const showNotification = ref(true);
</script>
`} />
