# SelectBtnField

Le composant `SelectBtnField` est utilisé pour permettre à l’utilisateur de sélectionner une valeur dans une liste.


<Canvas of={SelectBtnFieldStories.Default}  />

## API

<Controls of={SelectBtnFieldStories.Default} />

## Multiple

Vous pouvez permettre à l’utilisateur de sélectionner plusieurs valeurs en utilisant la prop multiple.
<Canvas of={SelectBtnFieldStories.Multiple}  />

## Item unique
Vous pouvez créer un bouton qui désélectionne tous les autres boutons en utilisant la prop unique.

<Canvas of={SelectBtnFieldStories.itemUnique}  />

## Inline
Vous pouvez afficher les boutons de sélection sur une seule ligne en utilisant la prop inline.

<Canvas of={SelectBtnFieldStories.inline}  />

## Messages d’aide
