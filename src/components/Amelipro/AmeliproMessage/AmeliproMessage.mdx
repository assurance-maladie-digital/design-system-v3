# AmeliproMessage

L’élément `AmeliproMessage` est utilisé pour afficher les différents messages d’alertes ou d’information.

Communiquer sur l’état d’une action est essentiel afin de guider l’utilisateur durant l’utilisation du portail et services. Pour cela différents types de messages sont à utiliser en fonction du type d’interaction. Ne jamais oublier que plus important que guider l’utilisateur lorsque une erreur se produit, c’est de faire en sorte de l’éviter.
