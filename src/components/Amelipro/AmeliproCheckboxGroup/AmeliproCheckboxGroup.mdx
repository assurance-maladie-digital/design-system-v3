# AmeliproCheckboxGroup

L’élément `AmeliproCheckboxGroup` est utilisé lorsque vous voulez donner la possibilité à l’utilisateur de sélectionner un certain nombre d’options. Cela comprend l’absence d’options, une seule option ou des options multiples. Les cases à cocher peuvent être utilisées également pour activer ou désactiver une option.
